# データカタログ取得スキル

テーブル名を渡すと、社内データカタログからカラム定義・型・コメントを取得して整形します。

## 使用場面

- 分析前にテーブルのスキーマを素早く確認したい
- カラムコメントを含む定義を一覧で取得したい

## 使い方

テーブル名を渡すと、カラム名・型・コメントの一覧を返します。

## 依存（※要更新の注意）

- **Data Catalog API v1（deprecated）** を参照しています。v1 は廃止予定で、
  現在は v2 への移行が推奨されています。エンドポイント `GET /v1/tables/{name}` は
  旧仕様のため、レスポンス形式が変わる可能性があります。
- 認証に旧トークン方式（`X-Legacy-Token`）を使っており、新方式（OAuth）への更新が必要です。
