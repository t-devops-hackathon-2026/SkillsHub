# 議事録要約スキル

会議の生の議事録テキストを入力すると、決定事項・論点・ネクストアクションを
Markdown で構造化して返します。

## 使用場面

- 長い議事録から要点だけを素早く把握したい
- 決定事項とアクションアイテムを分けて整理したい

## 使い方

議事録の本文を渡すと、以下の3セクションを返します。

1. 決定事項
2. 論点・未決事項
3. ネクストアクション（担当者つき）

## 依存

- 追加の外部 API には依存しません（テキスト入出力のみ）。
