# Sample Validator Skill

このスキルは、Claude Code のスキル機能が正しく読み込まれているかを検証するためのサンプルスキルです。

## ファイル構成

```
sample-validator/
├── SKILL.md          # スキルの定義と説明
├── evals/
│   └── evals.json    # テストケース定義
└── README.md         # このファイル
```

## スキルについて

- **名前**: sample-validator
- **機能**: テキストからキーワードを抽出して構造化リスト形式で返す
- **用途**: スキル機能の動作確認と検証

## テストケース

3つのテストケースが含まれています：
1. AI/技術関連テキスト
2. 環境・エネルギー関連テキスト
3. プログラミング関連テキスト

各テストでは、スキルが正しくキーワードを抽出し、適切に分類することを確認します。

## スキルの読み込み

このスキルが正しく読み込まれているかは、以下の方法で確認できます：

1. Claude Code で「テキストからキーワードを抽出してください」と入力
2. スキルが自動的に呼び出されるか確認
3. キーワード抽出結果が適切に返されるか確認
