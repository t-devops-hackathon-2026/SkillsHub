# テキストキーワード抽出スキル

テキストコンテンツを分析し、主要なキーワードを抽出して構造化されたリスト形式で整理するスキルです。

## 使用場面

- 長いテキストや記事から重要なキーワードを素早く抽出したい
- テキストの内容を簡潔にまとめる必要がある
- キーワードをカテゴリー別に整理して、内容を可視化したい

## 処理フロー

1. **テキストの受け取り** - ユーザーがテキストを提供
2. **キーワード抽出** - テキストから主要なキーワードを識別
3. **カテゴリー分類** - キーワードを適切なカテゴリー（技術用語、人物、場所など）に分類
4. **構造化出力** - マークダウンリスト形式で見やすく整理して提示

## 出力形式

```markdown
# テキスト分析結果

## 📌 主要キーワード
- キーワード1
- キーワード2

## 🏷️ カテゴリー別キーワード

### 技術用語
- 技術1
- 技術2

### 人物・組織
- 人物1
- 組織1

### その他
- キーワード1
```

## 指示のコツ

テキストキーワード抽出が効果的に動作するために：

- **明確さ** - 「このテキストから重要なキーワードを抽出してください」と明確に指示
- **フォーマット** - マークダウンリスト形式で整理することを明示
- **分類** - キーワードを適切にカテゴリー分けするよう指示
